# Changelog

## [0.1.0] - 2026-04-30

### Added
- TypeScript core library for repository analysis and README rendering.
- CLI commands: analyze, generate, serve.
- Fastify API with /health, /ready, /v1/analyze, /v1/generate.
- VS Code command integration: copilotForReadmes.generateReadme.
- Vitest suite (20 tests) covering analyzer, generator, template, and server.
- Dockerfile, docker-compose, and GitHub Actions CI with container smoke test.
- Governance docs: architecture, security, contributing, runbook.
